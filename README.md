## ΏθγΑ Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/mithushan-portfolio.git
cd mithushan-portfolio
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

The project should now be running locally.

For most React/Vite projects:

```bash
http://localhost:5173
```

For most Next.js projects:

```bash
http://localhost:3000
```
