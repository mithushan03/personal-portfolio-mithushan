## 📌 Overview

This project is a personal portfolio website created to build a strong online presence and showcase professional work.  
It can be used as a digital resume, project showcase, and contact gateway for recruiters, clients, collaborators, and visitors.

The website focuses on:

- Presenting Mithushan’s personal brand
- Highlighting technical skills and experience
- Showcasing projects and achievements
- Providing easy ways to connect
- Maintaining a clean, responsive, and professional UI

---

## ✨ Key Features

- **Responsive Design** — Works smoothly on desktop, tablet, and mobile devices.
- **Professional Landing Page** — Clear introduction and personal branding.
- **Portfolio/Projects Section** — Displays selected work and project highlights.
- **Skills Section** — Shows technologies, tools, and areas of expertise.
- **About Section** — Introduces background, interests, and professional goals.
- **Contact Section** — Makes it easy for visitors to reach out.
- **Fast User Experience** — Lightweight structure for quick loading.
- **SEO-Friendly Structure** — Suitable for search engine indexing and personal branding.
- **Clean Codebase** — Easy to maintain, customize, and extend.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/mithushan-portfolio.git
cd mithushan-portfolio
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

The project should now be running locally.

For most React/Vite projects:

```bash
http://localhost:5173
```

For most Next.js projects:

```bash
http://localhost:3000
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
